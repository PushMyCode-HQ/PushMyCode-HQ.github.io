The PushMyCode Workflow

1

Install & Connect Securely

Add the extension to Chrome. Click the extension icon and authenticate with your GitHub account. Choose an existing repository or let PushMyCode create a new one for you.

2

Choose Your Platform

Head over to LeetCode, HackerRank, GeeksforGeeks, or CodeChef. Start practicing your Data Structures and Algorithms.

3

Solve & Submit

Write your code and click the native "Submit" button on the website. No extra buttons to press, just use the platform normally.

4

Auto-Sync to GitHub

The moment your code gets "Accepted", PushMyCode instantly organizes the file, formats a Markdown explanation, and pushes it directly to your GitHub repository in the background.
